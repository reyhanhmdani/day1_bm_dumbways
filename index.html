Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web.

Struktur Dasar HTML

Struktur dasar HTML terdiri dari tag-tag seperti <html>, <head>, <body>, dan lainnya.
